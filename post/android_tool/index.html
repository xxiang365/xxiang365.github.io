Android下载V2rayNG以及导入Vmess链接

· 2020-11-02 · # Android # Tools

Step1: 下载并安装APP

安卓客户端可以使用V2rayNG
在github网址下载最新release版的安卓安装软件(apk格式的安装包)
#下载地址：
Android_V2rayNG github 网址

Step2: 导入服务端信息

打开app后，点击首页右上角的加号，可以看到有很多种方法导入配置信息，这里主要用到以下两种方法：

  • 通过导入包含服务器信息的Vmess链接
  • 通过扫描包含服务端信息的二维码

Step3: 设置app

导入完成后，还不能立即使用，原因是新版的app有个默认的选项和服务器端不一致，导致无法配置成功
需要修改一个地方：

  • 点红框中的按钮
  • 更改红框中的选项false改为true
