Windows使用v2rayN指南

· 2020-12-20 · # Windows
  1. 解压文件v2rayN-Core.zip到一个你记得住的路径，推荐C盘根目录
  2. 在解压之后的文件中打开.exe文件
  3. 从剪切板导入URL
  4. 右键选中，设置为活动服务器
  5. 设置开机启动
  6. 设置智能代理模式
    右键点击小托盘下的图标,
    并且设置Http代理模式为PAC模式.
    若有些网站不能上则选择 全局模式
    （用完改回PAC）
    Enjoy it!!!
    全局模式：所有流量走代理
    PAC模式：google，YouTube等黑名单的流量走代理，其余流量不走代理
