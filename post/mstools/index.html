Window 装机工具网站

· 2022-09-16 · # Windows

#镜像篇
##网站：你好Windows
地址：https://hellowindows.cn
评语：涵盖从XP到WIN11，分类明细，访问速度快，提供电驴和磁力链接（BT）两种下载方式，提供下载工具，激活工具，楼主最常用的镜像下载网站

##网站：UUP dump
地址：https://uupdump.net
评语：最大的特点是从 Windows 更新服务器下载，支持下载正式版，DEV ，Beta等不同通道的版本，原汁原味，新鲜速递

##网站：系统之家
地址：https://www.xitongzhijia.net
评语：GHOST版系统镜像下载基地，雨林木风，深度技术，番茄花园等各家版本都有，非原版控必备，顺便说一下优缺点，GHOST版本优点是部署快，缺点是捆绑很多软件，可根据需求选择使用原版还是GHOST版

#工具篇
##网站：优启通
地址：https://www.upe.net/download.html
评语：PE是系统预安装环境，用U盘做一个PE启动盘，以后安装和维护系统就无忧了，这个是楼主用过最好用的，其它像微PE什么的太多了，就不多推荐了

##网站：Office Tool Plus
地址：https://otp.landian.vip/en-us/
评语：最简单好用的Office下载软件，无广告，只可用于下载Office 工具，不可用于激活

#激活篇
##地址：http://webact.185.hk/sharedkey.php
评语：密匙有效性和激活次数一目了然。缺点：提供的版本不多，主要用来激活10/11，特别提醒，10和11的MAK是通用的哦

##网站: 云萌Windows10+激活工具
地址：https://cmwtat.cloudmoe.com/cn.html#
评语：云萌Windows10+激活工具，一键激活，开源透明且安全

##激活工具：HEU_KMS_Activator
地址：https://aichunjing.lanzoui.com/b710887
评语：支持多种版本，多种方式激活，用激活工具就是省时省力
